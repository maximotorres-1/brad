# Build the Bratwurst! (School Project Game)

A single-file HTML drag-and-drop game for a German school project.

## Live Game
- https://maximotorres-1.github.io/brad/brad.html

## Repository
- https://github.com/maximotorres-1/brad

## Features
- Drag or click ingredients to place on the plate
- Correct ingredients: sausage, curry powder, ketchup, oil
- Wrong ingredient feedback and score updates
- Timer + win condition
- Confetti and balloon celebration effects
- Background festive music with ON/OFF toggle
- Collapsible leaderboard (with shared online scores via Supabase)
- Mobile-friendly layout for phone screens

## Tech
- One self-contained file: `brad.html`
- Vanilla HTML, CSS, JavaScript
- Hosted on GitHub Pages

## Local Run
Open from any static server, for example:

```bash
python3 -m http.server 8000 --bind 127.0.0.1
```

Then visit:
- http://127.0.0.1:8000/brad.html
